# Book-Store

This project is created using technologies:    

### Frontend:    

- React v.18.2
- react-router-dom v.6.3.0
- Typescript
- Redux ToolKit
- React Final Form
- bootstrap

### Backend    
- Firebase-Authorization
- Firebase-FireStore Database

### Available Scripts

In the project directory, you can run:
